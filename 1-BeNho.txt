### 🛠️ TẦNG 2 — 7 MODULE (Bẻ nhỏ từng cái)

---

#### 🔥 Module 1 — AI Flashcard + Spaced Repetition

**Input:** File PDF hoặc text nhập tay  
**Output:** Bộ flashcard, hệ thống ôn tập thông minh

```
Việc cần làm:
├── UI upload PDF / nhập text
├── Gọi Gemini API → parse ra cặp [Câu hỏi / Câu trả lời]
├── Lưu flashcard vào DB (gắn với user)
├── Giao diện flip card (front/back)
├── Thuật toán Spaced Repetition (SM-2 algorithm)
│     → Dễ: ôn sau 7 ngày
│     → Trung bình: ôn sau 3 ngày  
│     → Khó: ôn lại ngay hôm sau
├── Thông báo nhắc ôn (email hoặc in-app)
└── Giới hạn Free: 20 flashcard/tháng
```

**DB cần:** `flashcard_sets`, `flashcards`, `review_schedule`

---

#### 🚀 Module 2 — Giải bài từng bước

**Input:** Câu hỏi nhập tay (text / ảnh chụp bài)  
**Output:** Lời giải chi tiết từng bước

```
Việc cần làm:
├── Ô nhập câu hỏi (text + upload ảnh)
├── Chọn môn học (Toán, Lý, Hóa, Anh, ...)
├── Gọi Gemini API với prompt được thiết kế giải từng bước
├── Render kết quả có định dạng đẹp (markdown + LaTeX cho Toán)
├── Nút "Giải lại" / "Hỏi thêm"
├── Lịch sử các bài đã hỏi
└── Giới hạn Free: 5 lần/ngày
```

**DB cần:** `solve_history`

---

#### 📚 Module 3 — Luyện đề + Chấm điểm tự động

**Input:** Đề có sẵn hoặc AI tạo đề  
**Output:** Điểm số + phân tích điểm yếu

```
Việc cần làm:
├── Kho đề (trắc nghiệm, theo môn/lớp)
├── Giao diện làm bài có đồng hồ đếm ngược
├── Submit → chấm điểm tự động
├── Màn hình kết quả: điểm, đáp án đúng, giải thích
├── Biểu đồ phân tích: chủ đề nào yếu nhất
├── AI tạo đề theo chủ đề (dùng Gemini)
└── Free: 3 đề/tháng | Premium: không giới hạn
```

**DB cần:** `exams`, `questions`, `user_attempts`, `answers`

---

#### 🎯 Module 4 — Micro-learning 5 phút/ngày

**Input:** Người dùng chọn chủ đề muốn học  
**Output:** Bài học ngắn mỗi ngày + streak tracking

```
Việc cần làm:
├── Onboarding: chọn chủ đề (Tiếng Anh, kỹ năng mềm, ...)
├── Hiển thị bài học ngắn hàng ngày (có thể pre-written hoặc AI gen)
├── Dạng bài: đọc + 1 câu quiz nhỏ cuối bài
├── Streak counter (chuỗi ngày học liên tiếp)
├── Reminder notification (email/in-app)
└── Premium: mở khoá toàn bộ kho bài học
```

**DB cần:** `lessons`, `user_streaks`, `lesson_progress`

---

#### 🧠 Module 5 — AI Note + Summary

**Input:** Paste text hoặc upload file PDF/DOCX  
**Output:** Tóm tắt + highlight ý chính + lưu note

```
Việc cần làm:
├── Ô nhập text lớn / upload file (Cloudinary)
├── Gọi Gemini API → tóm tắt theo dạng:
│     - Bullet points ý chính
│     - Tóm tắt 1 đoạn ngắn
│     - Highlight từ khóa quan trọng
├── Lưu note vào thư viện cá nhân
├── Tìm kiếm note đã lưu
└── Free: 3 lần/ngày | Premium: không giới hạn
```

**DB cần:** `notes`, `summaries`

---

#### 👨‍🏫 Module 6 — Tutor Marketplace

**Input:** Học sinh tìm gia sư / Gia sư đăng ký  
**Output:** Kết nối 2 bên, đặt lịch học

```
Việc cần làm:
├── Form đăng ký gia sư (môn, giá/giờ, lịch rảnh, CV)
├── Trang tìm kiếm gia sư (filter theo môn, giá, đánh giá)
├── Profile gia sư chi tiết
├── Đặt lịch học + thanh toán qua platform
├── Hệ thống đánh giá sau buổi học
├── Platform thu commission 10–15%
└── Phase 1: chỉ Toán + Tiếng Anh
```

**DB cần:** `tutors`, `bookings`, `reviews`, `payments`

---

#### 🧪 Module 7 — Quiz Generator

**Input:** Upload bài học hoặc nhập text  
**Output:** Bộ câu hỏi quiz tự động

```
Việc cần làm:
├── Upload nội dung (text / PDF)
├── Chọn: số câu hỏi, độ khó, dạng câu (trắc nghiệm / đúng-sai / điền vào chỗ trống)
├── Gọi Gemini API → tạo quiz
├── Preview và chỉnh sửa quiz trước khi lưu
├── Làm quiz ngay hoặc chia sẻ link
└── Free: 1 quiz/ngày, tối đa 10 câu | Premium: không giới hạn
```

**DB cần:** `quizzes`, `quiz_questions`, `quiz_attempts`
